# 青山商城

使用 `Flutter 2.x` 版本开发的一个 电子商城APP项目，支持 `Null Safety`

## Get Start

```sh
# 初始化项目资源包
flutter pub get

# 运行项目
flutter run
```

如果未能启动 `macos`, 则需先构建项目 `flutter create .`
